A state-of-the-art, premium portfolio showcasing my work as a **Full-Stack Developer** and **AI/ML Engineer**. The platform is designed with a "cyberpunk / mission control" aesthetic, utilizing high-fidelity glassmorphism, fluid micro-interactions, and a sleek dark mode.

## 🌟 Key Features

*   **Premium Aesthetic:** Obsidian & Gold design system with dynamic glassmorphism and animated glowing borders.
*   **AI Voice Integration:** Interactive Text-to-Speech (TTS) greeting feature in the hero section that introduces my profile.
*   **Immersive Micro-animations:** Tactile hover-lift effects, animated tech grids, floating glowing orbs, and scramble text effects.
*   **Interactive Certifications Gallery:** A responsive, interactive image grid that allows users to seamlessly browse and enlarge my professional credentials.
*   **Dynamic Project Stacking:** A beautifully crafted, sticky scrolling animation stack using `framer-motion` for displaying case studies and projects.
*   **Performance Optimized:** Utilizing React lazy loading and optimized radial gradients instead of heavy CSS blurs to ensure a stutter-free 60fps experience.

## 🛠️ Technology Stack

*   **Frontend Framework:** React (Vite) + TypeScript
*   **Styling:** Tailwind CSS + Vanilla CSS (for complex keyframe animations)
*   **Animations:** Framer Motion
*   **Deployment:** Vercel

## ⚙️ Getting Started Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/tahanawab4848/Port-Folio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd "Taha's Portfolio"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173`.
